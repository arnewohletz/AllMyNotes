36. Arrays
==========
* Arrays sind Objekte, die mehrere 'Fächer' enthalten
* Arrays selbst sind komplex, können jedoch einfache Datentypen enthalten
* Arrays können nur einen Datentyp enthalten

**Definition**

Besser: :java:`int[] intArray`

Nicht gut (aber möglich): :java:`int intArray[]`

Some test: :java:`execute();`

**Deklaration**

.. code-block:: java

    int[] intArray = new int[3];  // Array mit drei Fächern

Die Größe von Arrays lässt sich nach Deklarierung nicht mehr verändern.
