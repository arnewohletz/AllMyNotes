Vue.JS
======
Client-seitiges JS Framework für Single Page Application. Variante von MVC
(Model-View-Controller). Open Source basiert. Versucht, die Nachteile von React
oder Angular zu verbessern. Basiert wie React auf Komponenten. Es erlaubt die
Struktur der Anwendung anzupassen. Ebenfalls kaum Interaktion mit der DOM.
Beliebig viele Instanzen von Komponenten möglich.

Auch ohne NodeJS einsetzbar.

Siehe ``Tag 25/vue``.

Vue Applikation über NodeJS CLI erstellen:

.. code-block:: none

    npx @vue/cli create my-project

Die Applikation dann über:

.. code-block:: none

    npm run serve

starten.

Das Build-Skript (Auslieferversion ohne NodeJS) lässt sich über

.. code-block:: none

    npm run build

erstellen.
