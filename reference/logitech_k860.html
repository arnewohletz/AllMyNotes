Logitech K860 - Special keys [1]

The Logitech K860 allows to be switch between Windows and macOS layout using special key combinations. Hold these combinations for three seconds.

fn + o

swaps Windows layout to Mac layout

fn + p

swaps Mac layout to Windows layout

fn + u

swaps special characters like ‘#/^’ and ‘A/°’ with the ‘ >’ and ‘<’ keys *

fn + b

pause break

fn + ESC

swaps between the smart keys and F1-F12 keys

* This only affects European 102 and US International layouts. fn + u only works on Mac layouts, so make sure you have switched to Mac layout by pressing fn + o.

[1]

Logitech. Keyboards: special key combinations. 2024. URL: https://prosupport.logi.com/hc/en-us/articles/4407176154903-Keyboards-Special-Key-Combinations (visited on 2024-01-17).
