Files

Files

Command

Short description

cat / head / tail

text viewer: show all / first rows / last rows

chmod

Show / edit access rights

chown

Show / edit owner

cp

Copy file(s)

file

Get exact file type

find

Search for any file attribut

less / more

text viewer: page-by-page display

ln

Create soft and hard links for files and directories

locate

Index search for file names

ls

List directory content

md5sum

Determine checksum

mimetype

Determine file type

mv

Move / rename files

rdfind

Find / edit / link file duplicates

rm

Remove files and directories

shred / wipe

Save delete file

split

Split file in smaller parts

touch

Create new file

System

System

Command

Short description

hostname

Server name or computer name

env

Get current system variables (for example path, language encoding, user account)

getconf

Get POSIX configuration variables (usually only system architecture)

inxi

Very exhaustive info on system, network, package sources, hardware, devices, tasks (usually not preinstalled)

launchctl / systemctl

Service info and control

lsb_release

Get short distribution info

mount

Get all mounted drives (also network drives)

neofetch

Linux Mint only: get system data like system, hostname, kernel, uptime, CPU, GPU, RAM

rtcwake

Enter a system sleep state until specified wakeup time

shutdown / reboot

System shut down and restart

service / systmctl

Control active and deactivated services (systemctl is preferred on Ubuntu-like systems)

sysctl

Control Kernel-Parameters

uname

System & Kernel short info

uptime

Get system runtime

whereis

Get program path

which

Get program path

System with systemd

Command

Short description

bootctl

Detailed boot info

hostnamectl

Show and change computer name

journalctl

Print log entries from the systemd journal

localectl

Show and change system language

loginctl

Show/end logins and sessions

networkctl

Manage network adapters, IPs and MAC addresses

resolvectl

Resolve domain names, IPV4 and IPv6 addresses, DNS resource records and services introspect, and reconfigure the DNS resolver

systemctl

Control tool for services

timedatectl

Show/change time and time zone

systemd-analyze

Show exact time for system start

systemd-cgtop

CPU and RAM usage of system services

sytemd-mount

Show/create mount points

systemd-path

Show detailed commented system paths

Command

Short Description

iwlist

Search for Wi-Fi connections

net

nload

Network analysis

nmap

LAN & WAN port analysis

nmcli

Terminal console of the network manager

ping

rfkill

Display, activate or deactivate all wireless adapters (Bluetooth, Wi-Fi, UMTS)

route

Display system routing table and Gateway IP

scp

smbpasswd

sftp

Copy data from/onto SSH server

ssh

ssh-copy-id

SSH copy access key (client)

ssh-keygen

SSH key generator (client)

traceroute

wget

whois

Command

Short Description
