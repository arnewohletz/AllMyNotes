Coding fonts [1]

Important

To install these fonts via Homebrew, first run:

$ brew tap homebrew/cask-fonts

B612 Mono (https://b612-font.com/):

$ brew install font-b612

👍 👍 Iosevka Mono (https://github.com/be5invis/Iosevka):

$ brew install --cask font-iosevska

👍 Sudo (https://github.com/jenskutilek/sudo-font):

$ brew install --cask font-sudo
[1]

Philipp Ascany. Choosing the best coding font for programming. 2023. URL: https://realpython.com/coding-font/ (visited on 2023-09-07).
